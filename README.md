# 🛒 QuickMart - Grocery Delivery App

## 📌 About The Project

QuickMart is a **full-stack grocery delivery application** built with the MERN stack. It provides a seamless shopping experience with features like user authentication, product browsing, cart management, wishlist, and AI-powered search.

---

## 🛠️ Tech Stack

### Frontend
- **React.js** - UI Framework
- **React Router DOM** - Navigation
- **Axios** - HTTP Client
- **React Toastify** - Notifications
- **CSS3** - Styling

### Backend
- **Node.js** - Runtime Environment
- **Express.js** - Web Framework
- **MongoDB** - Database
- **JWT** - Authentication
- **Razorpay** - Payment Gateway
- **Google Gemini AI** - AI Search

### Admin Panel
- **React.js** - Admin Dashboard
- **Cloudinary** - Image Upload

---

## ✨ Features

### 👤 User Features
- JWT-based Authentication (Login/Register)
- Product Browsing with Categories
- Search Products with AI (Gemini API)
- Add to Cart / Remove from Cart
- Wishlist Management
- Order Placement and Tracking
- Profile Management

### 🛒 Cart & Orders
- Add/Remove items from cart
- Update quantities
- Place orders with address
- View order history
- Cancel orders

### 🎨 Admin Panel
- Dashboard with statistics
- Product Management (CRUD)
- User Management
- Order Management

### 💳 Payment
- Razorpay Payment Integration
- Cash on Delivery Option
